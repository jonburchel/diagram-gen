Long Description:
The diagram compares three Foundry agent types side by side. Prompt Agents require no code, are fully managed, handle single-agent orchestration, and are best for prototyping and simple tasks. Workflow Agents require no code (YAML optional), are fully managed, support multi-agent branching, and are best for multi-step automation. Hosted Agents require code, use container-based managed hosting, support custom logic, and are best for full control with custom frameworks. Arrows from "Simple needs," "Orchestration needs," and "Advanced needs" guide users to the appropriate agent type.
